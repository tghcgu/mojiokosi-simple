# 文字起こし

Windows 11 で再生しているデスクトップ音声を、文字だけの画面にリアルタイム表示します。

- 起動すると自動で文字起こしを開始
- 画面内のボタンは閉じる `×` だけ
- マイクは使わず、PC で再生中の音声だけを認識
- 音声と認識処理は PC 内で完結
- 結果は `transcripts` フォルダーにも自動保存

## 必要なもの

- Windows 11
- Windows の「ライブ キャプション」
- 日本語の音声認識パック

初めてライブ キャプションを使う場合だけ、Windows が表示する案内に従って言語ファイルをセットアップしてください。セットアップ後は、文字起こし画面だけが表示されます。

## セットアップ

このフォルダーを開いた状態で PowerShell を起動し、次を実行します。

```powershell
powershell.exe -NoProfile -ExecutionPolicy Bypass -File .\.app\Install-Shortcuts.ps1
```

デスクトップに `文字起こし` ショートカットが1つ作成されます。以前の開始・停止ショートカットがある場合は削除されます。

## 使い方

1. デスクトップの `文字起こし` を開く
2. PC で動画、会議、配信などの音声を再生する
3. 文字起こしが自動で画面に表示される

画面の操作ボタンは、右上の閉じる `×` だけです。

- 移動: 画面の外側の余白をドラッグ
- 終了: 右上の `×`、`Esc`、または `Alt + F4`

終了すると、裏で使っていた Windows ライブ キャプションも自動で閉じます。

## 保存先

文字起こし結果は次の形式で保存されます。

```text
transcripts/caption-20260719-120000-123.txt
```

## 補足

このソフトは Windows 11 のライブ キャプションを認識エンジンとして使います。起動のたびにライブ キャプションを新しく開始するため、マイク入力は既定でオフになり、Windows の既定の音声出力デバイスだけが対象になります。

詳しい仕様は [Microsoft のライブ キャプション案内](https://support.microsoft.com/ja-jp/accessibility/windows/use-live-captions-to-better-understand-audio) を参照してください。
